# Cybersecurity & Risk Management

## Build

**Establish a Security Baseline:** Define the minimum-security requirements for all company systems and data.

**Create Core Policies:** Develop a foundational set of security policies (e.g., Acceptable Use, Information Handling).

**Form a Security Team:** Designate individuals or a team responsible for leading and managing the cybersecurity program.

**Identify and Classify Data:** Understand what types of data you have and classify it based on sensitivity (e.g., Public, Internal, Confidential).

## Manage

**Promote Security Awareness:** Regularly train employees on how to spot phishing attacks and follow security best practices.

**Oversee the Security Program:** Hold regular meetings to review security posture, active risks, and progress on initiatives.

**Coordinate Security Efforts:** Ensure all security-related activities (like vulnerability management, incident response, etc.) are working together.

**Report to the Business:** Provide clear, simple reports to leadership about the state of the security program and its value to the business.

## Maturity

- [ ] Do you have a basic set of written security policies?
- [ ] Is there a person or team officially in charge of cybersecurity?
- [ ] Have you conducted any form of security awareness training for employees?
- [ ] Do you have a high-level understanding of your biggest security risks?
